# Homebrew Tap

Homebrew formulae for [kamilpajak](https://github.com/kamilpajak) projects.

## Installation

```bash
brew tap kamilpajak/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [heisenberg](https://github.com/kamilpajak/heisenberg) | AI-powered test failure analysis |

## Usage

```bash
# Install
brew install kamilpajak/tap/heisenberg

# Upgrade
brew upgrade heisenberg
```
